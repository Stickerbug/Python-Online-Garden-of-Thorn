# Garden of Thorn / 荆棘花园

一个双人对战卡牌游戏。

玩家通过“三选一”构筑 15 张卡组，管理能量与魔力两种资源，使用攻击、装备、技能、反制四类卡牌，率先将对手生命值降至 0 即获胜。项目使用 Godot 开发，目标是实现局域网联机对战。

## 游戏特色

**双资源系统**。能量每回合自动回复，魔力需通过卡牌积累，两种资源并行管理。

**“三选一”构筑**。从动态卡池中经 15 轮三选一确定牌组，配比建议为 6 张攻击、4 张技能、3 张装备、2 张反制。每局体验不同。

**四类卡牌博弈**。攻击牌造成直伤，装备牌提供持续效果，技能牌实现一次性战术，反制牌可在对手回合响应。

**多种机制**。中毒层数每回合衰减，灼烧层数不衰减，同名卡使用惩罚，放逐、虚无、吸附、精准等关键词共同构成决策空间。

**回合上限**。超过 20 回合未分胜负，剩余生命值高者获胜。

## 快速开始（开发者）

1. 安装 [Godot 4.x](https://godotengine.org/download/)。
2. 克隆本仓库：

```bash
git clone https://github.com/Stickerbug/Garden-of-Thorn.git
```

3. 使用 Godot 导入项目文件夹。
4. 按 F5 运行项目。

当前仍在开发早期，功能尚不完整。

## 如何贡献

任何形式的参与都欢迎：

- **测试与反馈**：试玩当前原型，报告 Bug 或提出平衡性建议。
- **卡牌设计**：提交新卡牌方案，需符合现有资源模型和机制。不少卡牌命名与风格受到游戏 florr.io 启发。
- **代码贡献**：认领 Issues 中的任务，或直接发起 Pull Request。目前任务较少，也可以加入 QQ 群参与讨论。
- **美术与 UI**：如果你擅长绘制或处理图片、使用 Photoshop，欢迎联系。

提交 Pull Request 前请确保代码可正常运行，并简要说明改动内容。
