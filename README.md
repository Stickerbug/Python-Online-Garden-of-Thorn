# Garden of Thorn / 荆棘花园

一个双人对战卡牌游戏。

玩家通过"三选一"构筑 15 张卡组，管理能量与魔力两种资源，使用攻击、装备、技能、反制四类卡牌，率先将对手生命值降至 0 即获胜。

本项目为 Python 联机版，基于 Web（Flask + Socket.IO）实现浏览器端图形界面与拖拽交互，支持局域网及互联网对战、大厅邀请、选牌、反制响应等完整流程。
原生 Godot 版仍在开发中，详见：[Garden of Thorn 的 Godot 版仓库](https://github.com/Stickerbug/Garden-of-Thorn)。

## 游戏特色

- 双资源系统：能量每回合自动回复，魔力需通过卡牌积累。
- "三选一"构筑：从动态卡池中经 15 轮选牌确定牌组，每局体验不同。
- 四类卡牌博弈：攻击牌造成直伤；装备牌提供持续效果；技能牌实现一次性战术；反制牌可在对手回合响应闪避、减伤或抵消技能。
- 回合外反制：出牌后对手可触发反制提示，选择合适反制牌打断你的行动。
- 中毒层数衰减、灼烧层数不衰减、同名卡使用惩罚、放逐、虚无、吸附、精准等关键词共同构成决策空间。
- 超过 20 回合按剩余生命值判胜。

## 快速开始

### 环境要求

- Python 3.6+
- 安装依赖：`pip install -r requirements.txt`

### 启动步骤

1. 克隆仓库：

```bash
git clone https://github.com/Stickerbug/Python-Online-Garden-of-Thorn.git
cd Python-Online-Garden-of-Thorn
```

1. 启动服务器：

```bash
python app.py
```

服务器启动后会显示监听地址和端口（默认 5000），并进入命令行管理界面，可输入 `help` 查看指令。

1. 在浏览器中打开 `http://服务器IP:5000` 即可进入游戏。

> **提示**：如果只有一台电脑，直接打开 `http://127.0.0.1:5000`，使用两个浏览器标签页即可本机测试。

### 游戏流程

1. 输入昵称，点击"进入大厅"。
2. 在大厅中选择一名在线玩家，点击"邀请对局"；对方接受后进入选牌阶段。
3. 双方轮流完成 15 轮选牌，每轮从 3 张中挑选 1 张，有 3 次重选机会。
4. 选牌结束后选择开局事件，然后自动开始对战。
5. 对战中将手牌拖拽至出牌区域打出；若对手触发反制，会弹出响应窗口供选择反制牌。
6. 点击"结束回合"切换操作权。
7. 游戏结束后可申请再战或返回大厅。

### 操作提示

- 卡牌上显示能量（E）和魔力（M）费用，费用不足或非自己回合时无法出牌。
- 反制牌（如泡泡、邪眼护符）在对手出牌后才会被提示使用。
- 装备牌使用后持续生效，可通过污水牌摧毁（某些装备不可摧毁）。
- 战斗日志位于界面中部，实时显示伤害、回复等信息。
- 支持明暗主题切换和多语言选择（设置中切换）。

## 文件说明

| 文件                     | 用途                                            |
| ---------------------- | --------------------------------------------- |
| `app.py`               | 游戏服务器（Flask + Socket.IO），处理登录、大厅、邀请、房间管理、对战逻辑 |
| `game_engine.py`       | 核心游戏规则引擎，负责回合流程、资源管理、伤害结算、中毒/灼烧、装备触发等         |
| `cards.py`             | 卡牌数据定义（CardDef、CardInstance）、牌池生成、选牌逻辑        |
| `static/css/style.css` | 游戏界面样式，响应式布局适配各种屏幕比例                          |
| `static/js/game.js`    | 游戏前端逻辑，UI 渲染、事件交互、多语言支持                       |
| `templates/index.html` | 游戏页面模板                                        |
| `requirements.txt`     | Python 依赖                                     |
